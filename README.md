# 社会薬学総論 Webページアプリ

## 内容

- 全102問
- 選択問題
- 記述・空欄問題
- 自動採点
- 正答率
- 採点回数
- 苦手・直近不正解
- 未採点
- ランダム出題
- 問題一覧
- 履歴の書き出し・読み込み
- PWA対応
- オフラインキャッシュ対応

## 使い方

このフォルダの中身をWebサーバーに置いて、`index.html` をSafariで開きます。

iPhoneではSafariで開いたあと、共有ボタンから「ホーム画面に追加」を選ぶとアプリ風に使えます。
