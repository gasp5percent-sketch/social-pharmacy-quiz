## R8セットの変更点
- 正誤問題は各文ごとに「正しい／誤り」を選択して採点できる形式に変換
- 空欄問題・分類問題・計算問題は可能な範囲で選択式に変換
- ICERなど一部は複数選択・単一選択で自動採点
- 純粋な図表・まとめページは自己採点カードとして残存
- PDF画像は折りたたみ表示で確認可能
- 問題文は行の途中で不自然に切れないように整形
